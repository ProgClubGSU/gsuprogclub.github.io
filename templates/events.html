Conway’s Game of Life

We will learn how to create a game called Conway’s game of life discovered by the renowned mathematician John Conway! This isn’t your standard one or two player game, but a zero player cellular automation that utilizes a two dimensional grid of square cells in each of which is either alive or dead. We will be covering how you can create your own simulation using Java! If you're curious to learn more about how the game works or the libraries we’ll be using to draw check out the links below.

Information about the game: Beginner's Guide Game of Life
Libraries that will be used: Basics of Java GUI, Java AWT (Abstract Windows Toolkit) Library
